# Composite

Composite is a simple Image Editor created in c++ 23 with ImGui, GLFW and ImFlowNode.

![](https://github.com/Cherrytree56567/Composite/blob/master/preview/preview.gif?raw=true)

## Goals
 - Execute function
 - More Nodes
 - AI Nodes
 - Faster Processing
 - Themes
 - Export

## Features
 - Color Balance Node
 - Input Node
 - Output Node
